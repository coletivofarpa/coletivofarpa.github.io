A enorme quantidade e variedade de dados existentes e novos dados gerados no mundo hoje são sem precedentes. À medida que este crescimento continua, o mesmo acontece com as oportunidades para que as organizações utilizem seus dados para criar a mudanças relevantes. A Farpa se dispõe em ajudar movimentos sociais e grupos de pesquisa engajados a identificar e capturar o maior valor de seus dados, e transformá-los em tomada de decisão.

Serviços oferecidos:
- Consultoria e extração de fontes de dados abertos;
- Higienização de dados;
- Enriquecimentode de dados - Cruzamento de dados de organizações com fontes de dados abertos;
- Dashboard de visualização de dados 
- Site aberto com dados de indicadores e análises
